# 夜行者：归藏 NightKeeper

> 中国风像素潜行追缉游戏 · 腾讯云 CodeBuddy 黑客松参赛作品

## 一、立意

某博物馆遭窃，国宝流散。玩家扮演"守夜人"——文物追缉干员，潜入古墓、拍卖行、私人收藏，在巡逻的守卫眼皮底下，将本属于这片土地的器物一件件请回家。

## 二、核心循环

```
进入关卡 → 潜行搜寻 → 拾取文物 → 避开巡逻 → 抵达撤离点 → 入库 → 解锁百科 → 下一关
```

- **失败惩罚**：被发现 = 本局丢失已拾取文物（容器内文物保留）
- **背包系统**：格子有限，逼迫玩家做"取舍"
- **撤离倒计时**：博物馆开馆前必须撤离

## 三、技术栈

- 引擎：**Phaser 3 + Vite (JS)**
- 部署：浏览器（赛题硬性要求）
- AI 接入：腾讯混元（TokenHub），用于
  - 文物百科卡动态生成
  - 局外"鉴定官"NPC 对话
  - 关卡叙事文案

## 四、本地运行

```bash
npm install
npm run dev
```

打开 http://localhost:5174 查看游戏。

## 五、目录结构

```
NightKeeper/
├── index.html
├── package.json
├── vite.config.js
├── public/              # 静态资源（素材包放这里）
└── src/
    ├── main.js          # 入口
    ├── scenes/          # 游戏场景
    │   ├── BootScene.js
    │   ├── TitleScene.js
    │   └── MuseumScene.js
    ├── data/            # 关卡 / 文物配置
    │   └── relics.js
    ├── systems/         # 玩法系统（视野/AI/背包）
    └── ui/              # UI 组件
```
